# Открытые вопросы

Вопросы, требующие решения владельца проекта. Формат: контекст → варианты → рекомендация исполнителя.

Жизненный цикл: после решения владельца суть переносится в спеку/ADR/бэклог, а вопрос **удаляется** — история решения остаётся в git (коммит ссылается на номер вопроса). Нумерация сквозная и не переиспользуется: вопросы №1–№6 закрыты (имя Berth, платформы, язык, флаги спеки, дерево в панелях, семантика Undock); следующий вопрос получает №7.

*Открытых вопросов нет.*
