## 1. Имя «IdeaDocking» для публичного пакета

«IDEA» — торговая марка JetBrains; для публичного NuGet-пакета имя рискованно. Отдельная техническая деталь: пространство имён `IdeaDocking.Avalonia` конфликтует по разрешению имён с корневым `Avalonia` (классическая боль библиотек `X.Avalonia`; лечится `global::`, но постоянно).
Варианты: (а) оставить как рабочее имя, переименовать перед публикацией; (б) выбрать нейтральное имя сейчас (дешевле всего — проект молодой).
