# ADR-0007. Имя Berth и схема пространств имён

Статус: принято (2026-07).

## Контекст

Рабочее имя IdeaDocking содержало «IDEA» — торговую марку JetBrains, рискованную для публичного пакета. Кроме того, пространство имён с сегментом `Avalonia` ломает разрешение ссылок `Avalonia.*` внутри такого namespace (вложенный `X.Avalonia` затеняет корневой `Avalonia`).

## Решение

Имя — **Berth** («швартовное место»: точная метафора шести слотов панелей). Свободно на NuGet (проверено 2026-07-07).

- Пакеты/сборки: `Berth.Core`, `Berth.Avalonia` — по конвенции экосистемы Avalonia.
- Пространства имён — без сегмента «Avalonia»: ядро `Berth`, UI-слой `Berth.Controls` (тесты: `Berth.Core.Tests`, `Berth.Controls.Tests`).

## Следствия

- Коллизия с корневым `Avalonia` исключена по построению — `global::` не нужен.
- Решение (`IdeaDocking.slnx`), демо-хосты и папка репозитория переименовываются отложенно (задача в бэклоге): делать это под открытой IDE владельца неудобно.
