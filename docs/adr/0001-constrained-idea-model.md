# ADR-0001. Ограниченная модель докинга по образцу IntelliJ IDEA

Статус: принято (2026-07).

## Контекст

Free-form докинг (AvalonDock, Dock) допускает произвольную вложенность контейнеров, из-за чего пространство состояний неограничено, а краевые случаи неисчислимы — на практике это источник хронических багов таких библиотек.

## Решение

Тул-панели существуют только в шести слотах (3 стороны × 2 группы) с пятью режимами отображения и инвариантом «не более одной открытой докированной панели на слот». Произвольная вложенность допускается только в документной зоне — и то в форме дерева сплитов с таб-группами в листьях (ADR-0005).

## Следствия

- Пространство состояний перечислимо; операции — проверяемые переходы; инварианты покрываются property-тестами.
- Поведенческим эталоном служит IntelliJ IDEA (открытые исходники intellij-community) — спорные места сверяемы.
- Сценарии вне модели (например, тул-панель как вкладка документа) сознательно не поддерживаются.
